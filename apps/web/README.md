# Web app (stub)

The web app source lives in **`reputraq/apps/web`**. This folder is a workspace entry only.

**To run the web app**, from the **monorepo root**:

```bash
cd /Users/darshan/Desktop/reputraqv3
pnpm dev:web
```

Or from this repo root:

```bash
pnpm dev:web
```

Or run directly from the app directory:

```bash
cd reputraq/apps/web
pnpm dev
```

Web app will be at http://localhost:3001
